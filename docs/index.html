BMS Gateway

Industrial Modbus TCP ↔ BACnet/IP protocol bridge.
Flash in under 90 seconds — no IDE, no drivers required.

BACnet/IP · Port 47808 Modbus TCP · Port 502 SubscribeCOV ReadPropertyMultiple Vendor ID 260 Live Log Viewer ISA 101 Web HMI
Installation Steps
  1. Connect your ESP32-S3 (8 MB flash minimum) via USB.
  2. Click Install below and select your COM port.
  3. Wait for the flash to complete — approximately 60–90 s.
  4. On first boot, join the AP BMS-Gateway-Config (pw: admin1234) and open http://192.168.4.1.

Requires Google Chrome or Microsoft Edge (WebSerial API).
